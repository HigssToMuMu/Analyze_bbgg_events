2. UNIVERSAL PROPERTIES (ALL 8 EVENTS)
================================================================================

The following properties hold for every single event — this uniformity is
itself suspicious:

  - dZ = 0.000 exactly for all events (diphoton vertex displacement from PV)
  - Zero leptons: n_electrons = n_muons = n_leptons = 0
  - All 6 jet-pairing scheme flags = 1 simultaneously
    (is_nonRes, is_nonResReg, is_nonResReg_DNNpair, is_nonResReg_vbfpair,
     is_Res, is_Res_DNNpair)
  - Degenerate jet pairing: the b-jet pair is identical across all 6 schemes
    (lead bjet = jet1 always, sublead bjet = jet2 or jet3)
  - dijet_charge = 0 for all events
  - MultiBDT_output_0 > 0.9996 (signal BDT saturated at category 0)
  - D_ttH > 0.9999 and D_qcd > 0.999 (anti-ttH and anti-QCD discriminants
    nearly 1.0)
  - alpha, beta, gamma discriminant inputs all < 3e-4
  - All b-jets pass Loose + Medium + Tight b-tag WPs
  - Near-perfect back-to-back diphoton-dijet topology:
    |DeltaPhi(gg,jj)| > 2.6 for all events (close to pi)
  - weight = 1 for all events (confirmed data, not MC)

27. SUMMARY OF ROOT CAUSES
================================================================================

The 8 anomalous events share a consistent pattern strongly suggesting they
are QCD multijet or ttbar events faking the HH->bbgg signature:

  1. FAKE PHOTONS: 5/8 events have photons failing cut-based ID (chIso or
     H/E). 4/8 have conversion tracks, pixel seeds, or electron matches.
     The dominant failure mode is charged hadron isolation, indicating nearby
     hadronic activity or jet fragmentation products.

  2. ANOMALOUS B-JETS: High EM fractions (>30%) in many b-jets suggest
     photon or electron contamination within the jet cone. Extreme PNet
     regression corrections (40-50%) for some sublead b-jets indicate
     poorly measured jet energies.

  3. BDT SATURATION: The multiclass BDT assigns >99.96% signal probability.
     These events likely occupy a region of feature space (extreme m_HH
     tail, 452-697 GeV) where the BDT training statistics are sparse,
     leading to overconfident misclassification.

  4. DEGENERATE JET PAIRING: With effectively only 2 real b-jets per event,
     all 6 pairing schemes converge to the same answer. This inflates the
     event count — the same physical event appears in all scheme categories.

  5. HIGH MET (Events 1, 6): MET = 97-107 GeV aligned with jets, suggestive
     of ttbar with neutrinos from W->lv where the lepton is lost or merged
     with a photon candidate.

  6. BACK-TO-BACK TOPOLOGY + HIGH m_HH: The near-perfect DPhi~pi geometry
     and m_HH in the deep tail (452-697 GeV) is more characteristic of
     QCD dijet production than resonant HH.


================================================================================
28. RECOMMENDATIONS
================================================================================

  1. PHOTON ID: Require cutBased >= 1 (at least Loose) on BOTH photon
     candidates. This would remove Events 2, 3, 4 immediately.

  2. CONVERSION VETO: Veto events where hasConversionTracks=1 AND
     pixelSeed=1 on the same photon. This targets Events 4 and 6.

  3. ELECTRON VETO: Veto photons with electronIdx >= 0 (matched electron).
     This targets Events 3 and 6.

  4. BDT RETRAINING: Investigate BDT behavior in the high m_HH tail
     (>400 GeV). Consider adding m_HH-dependent corrections or retraining
     with enriched samples in this region.

  5. MET CUT: Consider a MET < 80 GeV cut to suppress ttbar. This removes
     Events 1 and 6.

  6. EM FRACTION: Consider flagging or vetoing b-jets with EM fraction
     > 40%, which would target Events 1, 4, 5.

  7. TRACK ISOLATION: Apply a trkSumPtHollowConeDR03 < 5 GeV cut on
     photons. This removes Events 2 and 3.

  Applying recommendations 1+2+3 together would remove 5/8 events
  (Events 2, 3, 4, 6, and potentially more). Adding the MET cut would
  also remove Event 1, leaving only Events 0, 5, 7 — which have the
  mildest pathologies.
